    Js, HTML and CSS

    Javascript is the brains behind HTML and CSS. It makes changes to the page like changing numbers or making buttons work

    Control Flow and Loops

    Control Flow acts like a routine for computer programs. Like a fitness routine, control flow tells the computer what order to do the actions in. Bench press first, then squats, then sit ups. For loops tell the computer how many times it has to perform an action, 10 sets on the bench press 3 times.

    The DOM

    The DOM is simply the way that HTML is organized so that the computer can take, add and change parts of the page (rather than doing it manually).
    It uses a tree structure to access different elements, each of which have a parent-child relationship. Think of a family tree where a container is the parent of a sub-container is the parent of the paragraph in that container

    Arrays vs Objects

    To get data from an Array, you have to grab it based on where it is in the array. An object can have an element selected by calling the object.element

    Functions

    Functions are small chunks of code that can be used time and again instead of having to write code out each time you want to use them. You only have to write the code once, then you just call the function when you want to use it
