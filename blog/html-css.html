    What are the differences between relative, absolute, and fixed positioning?

    Let's first start with some definitions:
    1. Position: Relative.Element is positioned relative to its normal position

    2. Position: Absolute.Element is positioned relative to the nearest positioned ancestor

    3. Position: Fixed. is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled
    How to Use them:

    Relative position can be used to line elements that are difficult to line up with absolute positioning
    Absolute Position is very useful when placing elements independent from each other. It can need some thought because it can disrupt the flow of the site
    Fixed Position is really great for when you need something to stick to the top or bottom of the page regardless of scrolling such as a nav or a footer
